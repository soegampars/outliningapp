# Keyboard shortcuts

| Key | Action |
|---|---|
| `Ctrl+S` / `Ctrl+O` / `Ctrl+N` | Save / Open / New project |
| `Ctrl+D` | Duplicate the selected node |
| `Delete` / `Backspace` | Remove the selection (nodes and/or edges) |
| `Esc` | Clear selection · exit a block (drill up) · cancel a dialog |
| Double-click empty canvas | Add a node |
| Double-click a node | Edit its claim (or drill into a block) |
| Double-click an edge | Toggle all-of / any-of |
| Left-drag on empty canvas | Marquee multi-select (selected nodes move together) |
| Right-drag, or Space-drag | Pan the canvas |
| Scroll | Zoom |
| Drag a support's grip (⠿) | Reorder supports in the inspector |

Shortcuts that work even while typing in a field: `Ctrl+S`, `Ctrl+O`, `Ctrl+N`.
The others apply when the canvas (not a text field) has focus.
